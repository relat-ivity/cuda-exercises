# cuda优化记录

### 规约算子
1. 交错规约，相邻的线程计算相邻数据，**减少分化**
2. 使用**共享内存**，存储每个block负责规约的数据
3. 对于最后64个要规约的数据，**展开循环**，在同一个warp里算完
4. 1个**线程展开**计算原本8个线程的数据

### 转置算子

1. 使用**共享内存**按行读入一个block要转置的矩阵块
2. 转置时候**交换**块内线程的x和y维度id，让转置过程也按行写入
3. 转置过程是按列读共享内存，需要使用padding**解决bank conflict**
4. 1个**线程展开**转置原本4个线程的数据，block大小变为(32, 8)
